# Larrow::Runner

a runner which build your application

## Installation

Add this line to your application's Gemfile:

    gem 'larrow-runner'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install larrow-runner

## Usage

larrow <source\_url>
